Политика конфиденциальности

1. Какие сведения о себе клиент должен указать, регистрируясь на сайте?

Начиная взаимодействовать с компанией, пользователи вносят в форму для регистрации на портале свой электронный адрес почты.

2. Как эта информация может быть использована?

Данные пользователей компания применяет, чтобы:

  • отдел технической поддержки работал качественнее;
  • высылать на электронную почту пользователей новую информацию об услугах организации;
  • своевременно реагировать и устранять действия мошенников.

3. Кто может иметь право допуска к информации клиентов, являющейся конфиденциальной?

Все данные пользователей, оставляемые ими на сайте при регистрации - не разглашаемы. При этом есть ряд исключений: разглашение информации по постановлению суда, запросу правоохранительных органов, при продаже или реорганизации компании.

4. Как используют файлы cookie?

4.1 Для чего нужны текстовые файлы cookie?

Файлы cookie помнят все, что делает пользователь на сайтах, и хранят эту информацию на компьютерном устройстве. Используя эти файлы, работники портала имеют сведения о сфере интересов посетителей, анализируя выбранные ими параметры. Таким образом, руководство сайта подстраивает контент в соответствии с предпочтениями пользователей, делая сотрудничество с ними более удобным.

4.2 Сколько времени файлы cookie могут храниться на компьютере?

Время хранения текстовых файлов на компьютере зависит от их вида. Но в любом случае оно не может быть больше периода, необходимого для завершения конкретных задач. После того как этот период будет завершен, данные из компьютера автоматически исчезают.

5. Гарантия безопасности данных клиентов.

Сотрудниками портала разработаны определенные гарантии защиты персональных сведений клиентов, способствующие уменьшению вероятности взлома мошенниками учетных записей. Тем не менее из-за быстрого развития технологий полной уверенности в защите нет. Клиенты должны следовать рекомендациям по использованию только сложных паролей, их надежному хранению и неразглашению другим лицам.
